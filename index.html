📖 Badminton Match Maker Pro

Version 4.6 - Official ReadMe & Manual

🌟 コンセプトと哲学

本アプリケーションは、スポーツコミュニティ運営において永遠の課題である「実力差の不均衡」と「休み回数の不平等」を全自動で解決し、幹事の負担を極限まで削ぎ落とすために開発された次世代のインフラツールです。

✨ 主要機能 (Core Features)

🤖 動的イロレーティングとシナジー解析

試合のスコアを入力するだけで、AIが「個人の実力（Rating）」と「ペアの相性（Synergy）」を分離して学習。相性の良し悪しを数値化し、常に白熱する最適なマッチアップを自動生成します。

⚖️ 完全な公平性と遅刻者補正

待機回数の多いメンバーを優先してコートへ配置。途中参加の遅刻者には自動でペナルティ（最小試合数への引き上げ）を適用し、不公平感を完全に排除します。

🌍 現場主義のオフライン・ファースト

電波の悪い体育館でもサクサク動くローカル設計。QR決済の連携や、急な休憩に対応する手動での「メンバー変更機能」など、現場のリアルな運用に寄り添っています。

🚀 クイックスタート (使い方)

  1. 事前準備: 「クラブ・練習会管理」タブで決済用QRコード等を設定し、「名簿管理」でメンバーを登録します。
  2. 出欠確認: 「今日の練習」タブで練習会を選択。メンバーをタップして「出席」にし、未納/集金済を管理します。
  3. 試合生成: 「最初の組み合わせを生成」を押し、Game 1&2を作ります。以降は「次の組み合わせ」で追加します。
  4. 結果入力: 試合後、得点を入れて「結果登録」を押すと、個人のランクと相性データが自動更新されます。
